Container alignment

Lorem ipsum. Short description of aligning containers. Mention of difference between using column() argument and helper function.

$container-align:right

Arguments

$align: ""
left|right|center

Example

section {
  @include columns((1,2), $container-align: "center");
  
  @media #{$from-desktop} {
    @include container-align("right");
  }
}
